# 설치 사용법

## 소스코드 다운로드

```
git clone https://github.com/jeongjuwon/koreauniv.crimson.git
```

## 앱 실행 방법

```
cd CrimsonProjet
npm install
npx pod-install
npx react-native run-ios
npx react-native run-android
```

## 서버 실행 방법

```
cd CrimsonServer
npm install
DEBUG=crimsonserver:* npm start
```
